# Tarea 1 - Programacion II

## Compilar y ejecutar cada reto

- Cada comando debe ser ejecutado en la carpeta de cada reto

### Reto 1: Empleado
```bash
javac MainEmpleado.java && java MainEmpleado
```

### Reto 2: Cuenta Bancaria
```bash
javac MainCuentaBancaria.java && java MainCuentaBancaria
```

### Reto 3: Inventario
```bash
javac MainInventario.java && java MainInventario
```

### Reto 4: Jugador de Rol
```bash
javac MainJugador.java && java MainJugador
```

### Reto 5: Aula Estudiantil
```bash
javac MainAula.java && java MainAula
```

## Estructura de cada reto

- `MainX.java` - Clase principal con metodo main
- `X.java` - Modelo con atributos, getters, setters y constructores
- `ControladorX.java` - Orquestador de las operaciones
- `DatosX.java` - Datos de prueba
